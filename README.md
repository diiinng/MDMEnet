## 简介
这是一个用于生活垃圾分类的深度学习模型，基于多模型级联卷积神经网络（MCCNN）。

## 功能特性
- 支持 52 种生活垃圾类别的检测与分类。
- 结合 DSSD、YOLOv4 和 Faster-RCNN 模型，提升检测精度。
